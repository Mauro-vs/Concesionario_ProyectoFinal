Validamos los campos de un modelo de moto para asegurarnos de que cumplen con las reglas de negocio antes de agregarlo o actualizarlo en la base de datos. Esto incluye validar que los campos obligatorios estén presentes, que los valores numéricos sean razonables, y que la matrícula tenga un formato correcto y sea única si se proporciona.
